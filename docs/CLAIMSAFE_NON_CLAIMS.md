# ClaimSafe Non-Claims

ClaimSafe does not claim:

- that an AI model is accurate
- that a benchmark is sufficient for every deployment context
- that a strategy, agent, or system will work in production
- that future returns or future behavior are guaranteed
- that a release is legally, medically, financially, or operationally approved

ClaimSafe v0.1 claims only that a packet can be checked for bounded claim scope, required evidence, file hashes, replay status, release gate status, and preserved non-claims.

Good wording:

> This project verifies whether an AI research claim has bounded evidence, reproducible gates, and preserved non-claims.

Unsafe wording:

> This AI finds winning stocks.
